Politique de confidentialité

LUCID — Jeu de puzzle hexagonal
Éditeur : Californy Lana

Résumé

LUCID ne collecte aucune donnée personnelle.
Le jeu fonctionne entièrement hors ligne. Aucune information n'est transmise à un serveur, un tiers ou l'éditeur.

Données stockées localement

LUCID enregistre uniquement des données de gameplay sur votre appareil :

  • Préférences : son (on/off), vibrations (on/off), langue
  • Progression : leçons complétées, niveaux débloqués
  • Statistiques : parties jouées, victoires, meilleurs scores, tours, temps par niveau
  • Mode chaîne : sauvegarde de la partie en cours, meilleur run

Ces données sont stockées localement via le système PlayerPrefs de Unity et ne quittent jamais votre appareil.

Aucune collecte de données personnelles

LUCID ne collecte, ne stocke et ne transmet aucune donnée personnelle, notamment :

  • Aucun nom, email ou identifiant
  • Aucune donnée de localisation
  • Aucun identifiant publicitaire
  • Aucune donnée d'utilisation transmise à distance

Aucun accès réseau

Le jeu ne nécessite pas de connexion Internet et ne communique avec aucun serveur. Il n'intègre aucun SDK d'analytics, de publicité ou de suivi.

Suppression des données

Vous pouvez supprimer toutes les données enregistrées à tout moment :

  • Dans le jeu : Statistiques → Effacer mes données
  • En désinstallant l'application (supprime automatiquement toutes les données locales)

Enfants

LUCID ne collectant aucune donnée personnelle, il est compatible avec une utilisation par des enfants sans préoccupation liée à la vie privée.

Contact

Pour toute question : californylana@gmail.com

Privacy Policy

LUCID — Hexagonal puzzle game
Publisher: Californy Lana

Summary

LUCID does not collect any personal data.
The game works entirely offline. No information is sent to any server, third party, or the publisher.

Data stored locally

LUCID only saves gameplay data on your device:

  • Preferences: sound (on/off), haptics (on/off), language
  • Progression: completed lessons, unlocked levels
  • Statistics: games played, wins, best scores, turns, times per level
  • Chain mode: current run save, best run

This data is stored locally using Unity's PlayerPrefs system and never leaves your device.

No personal data collection

LUCID does not collect, store, or transmit any personal data, including:

  • No names, emails, or identifiers
  • No location data
  • No advertising identifiers
  • No remotely transmitted usage data

No network access

The game does not require an Internet connection and does not communicate with any server. It does not include any analytics, advertising, or tracking SDK.

Data deletion

You can delete all saved data at any time:

  • In-game: Statistics → Delete my data
  • By uninstalling the app (automatically removes all local data)

Children

Since LUCID does not collect any personal data, it is safe for use by children with no privacy concerns.

Contact
